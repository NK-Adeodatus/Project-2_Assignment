# C Projects Repository

This repository contains three C projects. Each project is a separate program with its own functionality.

## Files

- **project1.c** – Dynamic Student Record Engine: stores, analyzes, and processes student data.  
- **project2.c** – Adaptive Text Intelligence Tool: analyzes a user-entered paragraph and provides word statistics.  
- **project3.c** – Callback-Based Device Monitoring Simulator: simulates devices sending readings processed via callback functions.  

## How to Run

1. Compile the desired project using a C compiler:

   ```bash
   gcc project1.c -o project1
   gcc project2.c -o project2
   gcc project3.c -o project3
